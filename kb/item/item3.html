Old Navy Redesign

Project Description

This project explored the process of utilizing a mobile interface to check out goods online. I analyzed Old Navy's checkout process and found that it failed to take advantage of mobile design principles. I then redesigned it to create the optimal checkout experience.

Pictured to the left: The redesigned Old Navy checkout focuses on the bare minimum of features necessary to completing the purchase. The existing Old Navy mobile interface is packed with advertisements and distractions, making for an overwhelming design overall. I focused on not only eliminating these unnecessary elements, but also implementing features that make for a smoother mobile checkout in order to secure the most possible sales.

This was a project for Advanced Principles of Interactivity, a course at DePaul University, and is not affiliated with Old Navy.
