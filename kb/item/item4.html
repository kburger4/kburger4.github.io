Grubhub User Research and Analysis Project

Project Description

This project involved researching Grubhub's target user demographic and evaluating whether it effectively met their needs. In order to do this, we started by critiquing the heuristics used in the Grubhub web interface and layed out the 'happy path', or path of least resistence to the user goal - in this case: ordering food. We then researched Grubhub's target users, created personas, and gathered users for user testing. Our findings are outlined in the documents below.

Project Details

  • User research, stories, and personas
  • Heuristic evaluation
  • User testing
